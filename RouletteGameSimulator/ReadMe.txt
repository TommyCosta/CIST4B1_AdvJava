Project Name: Roulette Game Simulator

Description:
This program is a roulette game simulator that is meant to mirror a real life casino experience entirely through the user's console.

Files Included:
- RouletteGame.java
- Decision Tree.java
	- NodeTree.java
- PayoutQueue.java
	- NodeBet.java
- SpinStack.java
- TableBet.java
	-TableBetEntry.java
- Wheel.java
	- WheelResult.java
- Bet.java
	- BetBasket.java
	- BetColor.java
	- BetColumn.java
	- BetCorner.java
	- BetDozen.java
	- BetEvenOdd.java
	- BetHighLow.java
	- BetLine.java
	- BetSplit.java
	- BetStraight.java
	- BetStreet.java
-ReadMe.txt

How to Compile:
Open a terminal in the project folder and run:
javac *.java

How to Run:
After compiling, run:
java RouletteGame
